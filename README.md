# Guides 3ADK

Ressources techniques produites par [3ADK Agency](https://3adkagency.fr) — guides pratiques autonomes pour maîtriser les outils IA et la création de contenu vidéo.

---

## Guides disponibles

### Claude Code Mastery Guide
**Fichier :** `claude-code-mastery-guide-3adk.html`

Guide complet pour utiliser Claude Code (l'outil CLI d'Anthropic) en contexte professionnel et agence. Couvre la configuration, les workflows avancés, l'intégration dans une stack d'automatisation.

### Video Content Factory
**Fichier :** `video-content-factory-3adk.html`

Système clé-en-main pour produire du contenu vidéo en série avec l'IA. Architecture de prompt, pipelines de génération, stratégie de publication.

---

## Utilisation

Les guides sont des fichiers HTML standalone — aucune dépendance, aucune installation requise.

```bash
# Ouvrir directement dans le navigateur
open claude-code-mastery-guide-3adk.html
# ou double-clic sur le fichier
```

Compatibles avec tous les navigateurs modernes.
